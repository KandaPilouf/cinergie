# Hackaton 2026: projet Cinergie
Dans le cadre de ce Hackaton 2026 organisé par l'EFP, nous avons travaillé sur la base du code du site internet de Cinergie.
Nous avons scrap le code des pages qu'il nous était demandé de modifier en gardant la charte graphique de Cinergie.
nous avons privilégié le mobile first tout en gardant le mode desktop cohérent

### Général
-liens vers les pages modifiées fonctionnels (navbar et footer)
-responsive

### Homepage
-refonte des cards
-side scroll en format mobile
- reprise des cards de l'agenda pour les rendez-vous avec le cinéma Belge

### Agenda
-refonte complète de l'agenda sur un style reppelant les affiches de cinéma
-utilisation du grid

### Articles
-restructuration de l'article
-reprise du side scroll pour les contenus liés et autres articles

### Boutique
-refonte totale de la boutique

### Notre histoire
-restructuration de la page histoire avec des bulles qui illustrent les années
